Paragraphs

.p and .p1

I'm going to style paragraphs the same way I style headers. As such, I want to keep the naming convention the same. However, I've already started the habit of just typing out the <p>. So I don't want to get penalized for doing so. This is the kind of body copy that I want to use when I just want my writing to have some simple formatting. The goal of this class is easy reading.

.p2

Adding a <p2> is the second most used style. I think this kind might be used for a gist of sorts

.p3

This is probably when I want to make a big, bold statement but don't really feel like this is ideal for a headline. I mean, look how much text is in here. Bold, yet readable.
